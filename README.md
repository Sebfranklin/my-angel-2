# My Angel 2

A beautiful, minimalistic, animated love letter website for Sonia.

## Tech Stack
* HTML5
* CSS3 (Mobile-First responsive layout)
* JavaScript
* GSAP (GreenSock Animation Platform) & ScrollTrigger for high-performance animations
* HTML5 Canvas for twinkling night sky and shooting stars

## Hosting Locally
To host and preview this site locally (avoiding mobile CORS restrictions on local asset loading):
```bash
python3 -m http.server 8080
```
Then open `http://localhost:8080` in your web browser.
